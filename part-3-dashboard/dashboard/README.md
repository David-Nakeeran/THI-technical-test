# THI Technical Test - Part 3

This project is a dashboard for visualising survey results, focusing on numerical scores and comments. It was built as part of the technical test for The Happiness Index.

## Features

- **Filterable Results:** View survey data by question topic, department, and location.
- **Numerical Table:** Shows average, minimum, maximum scores, and response counts for each question.
- **Bar Chart:** Visualises minimum, average, and maximum scores per question with a highlighted neutral line.
- **Comments Accordion:** Displays survey comments for each question topic, expandable/collapsible.
- **Clear Filters:** Reset all selected filters quickly.

## Tech Stack

- **Frontend:** React, Next.js
- **Styling:** Tailwind CSS
- **Charts:** Chart.js, chartjs-plugin-annotation
- **Data:** JSON file used as mock survey dataset
